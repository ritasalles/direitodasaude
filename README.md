# Rita Salles | Direito da Saúde

Site profissional para a advogada Rita Salles, especialista em Direito da Saúde.

## 🌐 Sobre o Site

Este é um site moderno e responsivo desenvolvido com React 19, Tailwind CSS 4 e TypeScript. O site apresenta:

- **Hero Section**: Apresentação profissional com foto da advogada
- **Nossa Experiência**: Casos de sucesso com resultados reais
- **Teses Jurídicas**: Fundamentação legal consolidada
- **FAQ**: Perguntas frequentes sobre Direito da Saúde
- **Contato**: Formulário de contato e dados clicáveis (telefone, WhatsApp, e-mail)
- **SEO Otimizado**: Meta tags, schema.org e Open Graph

## 📋 Funcionalidades

- ✅ Logotipo de alta resolução
- ✅ Dados de contato clicáveis (tel:, mailto:, WhatsApp)
- ✅ Seção de FAQ expansível
- ✅ Design responsivo (mobile, tablet, desktop)
- ✅ Otimizações de SEO
- ✅ Tipografia elegante (Playfair Display + Lato)
- ✅ Paleta de cores profissional (Gold, Navy, Cream)

## 🛠️ Tecnologias

- **Frontend**: React 19, TypeScript, Tailwind CSS 4
- **Build**: Vite
- **Componentes**: shadcn/ui
- **Hospedagem**: Vercel

## 📦 Instalação

```bash
# Instalar dependências
pnpm install

# Iniciar servidor de desenvolvimento
pnpm dev

# Build para produção
pnpm build

# Preview da build
pnpm preview
```

## 🚀 Deploy

O site está configurado para deploy automático na Vercel. Qualquer push para a branch `main` acionará um novo deploy.

### Configurar domínio customizado

1. Acesse o painel da Vercel
2. Vá para Settings > Domains
3. Adicione o domínio: `direitodasaude.ritasallesadvocacia.com.br`
4. Configure os DNS records conforme instruído
